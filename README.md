# 🚀 React Authentication UI Project

## 📌 Overview
This is a simple React-based web application that includes a **Welcome Page**, **Login Page**, **Create Account Page**, and a **Profile Page**. The project demonstrates basic frontend routing and user authentication flow UI using React.


## ✨ Features

- 🏠 Welcome Page with navigation options
- 🔐 Login Page for existing users
- 📝 Create Account (Signup) Page for new users
- 👤 Profile Page displayed after successful login/signup
- 🔄 Simple navigation between pages using React Router
- 📱 Responsive and clean UI design



## 🛠️ Tech Stack

- React.js (Frontend Library)
- React Router DOM (Navigation)
- HTML5 & CSS3
- JavaScript (ES6+)


## 🔑 Workflow
  - User lands on Welcome Page
  - Chooses either Login or Create Account
  - After successful login/signup → redirected to Profile Page
  - Profile page displays user details (basic UI level)
